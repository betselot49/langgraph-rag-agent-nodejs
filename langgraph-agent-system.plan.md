```

├── src/
```

### Step 2: Weaviate Database Setup

**Goal**: Docker containerize Weaviate and verify connectivity

**Actions**:

- Create `docker-compose.yml`:
  ```yaml
  services:
    weaviate:
      image: semitechnologies/weaviate:1.27.0
      container_name: weaviate
      ports:
  - "8080:8080"
      environment:
        AUTHENTICATION_ANONYMOUS_ACCESS_ENABLED: 'true'
        PERSISTENCE_DATA_PATH: '/var/lib/weaviate'
        DEFAULT_VECTORIZER_MODULE: 'none'
        ENABLE_MODULES: ''
      volumes:
  - weaviate_data:/var/lib/weaviate
  volumes:
    weaviate_data:
  ```

- Create `src/database/check-connection.js` to verify Weaviate is running
- Add npm script: `"check-db": "node src/database/check-connection.js"`
